## How to Run

1. Make sure you have Node.js installed. I use npm version 11.6.1 and Node.js version 24.11.0 for development
2. Run `npm install -g serve`.
3. Run `npm start`.
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
